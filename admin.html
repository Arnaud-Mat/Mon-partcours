Espace privé

Cette page sert à modifier la timeline de ton site. Elle n'est volontairement liée nulle part publiquement.

Ceci ne bloque qu'un visiteur occasionnel. La vraie sécurité vient ensuite, avec ton token GitHub personnel que toi seul possèdes.

Édition de la timeline

Les modifications sont publiées directement sur data/projects.json dans ton repo GitHub. Ton token n'est jamais envoyé ailleurs qu'à l'API GitHub, et n'est conservé que pour la durée de cet onglet (sessionStorage, effacé à la fermeture).

Connexion GitHub

Crée-le sur github.com → Settings → Developer settings → Fine-grained tokens. Limite-le à ce seul repo et au scope "Contents: Read and write". Ne le partage jamais, ne le colle nulle part ailleurs.
